01 · 보안 공부

간호사에서 보안으로

중환자실 경력을 포함한 만 3년 6개월의 간호사 경력을 쌓은 후, 경력과 기술력이 공부한만큼 드러난다는 확신과 함께 보안으로의 이직을 결심했습니다.

약 1년 6개월간 보안 기초 지식을 쌓으며 준비했습니다:

  • GNS 가상 네트워크에 라우터 이미지 설치 후 RIP~BGP 네트워크 프로토콜 통신 실습
  • Linux(Ubuntu, CentOS) + Windows Server 다중 환경 구성 및 FTP, DNS, DHCP, WEB/WAS, DB 서버 구축
  • DVWA, Beebox를 통한 OWASP Top 10, SQL Injection 등 웹 취약점 실습
  • Dos/DDoS, Sniffing, ARP Spoofing, DNS Spoofing, Meta Exploit 해킹 기술 실습
02 · 국립암센터

R&D 보안 과제 연구원

2019년 8월, 의료기관 연구시스템의 보안 취약점 연구를 위해 국립암센터 R&D 과제 연구원으로 입사하였습니다.

과제명: "기관 간 분산형 CDM(Common Data Model) 환경의 보안 취약점 분석 및 진단 Tool 개발"

  • 5억 규모 R&D 과제를 100% 참여로 주도적 수행
  • 600억 5년 과제 "암빅데이터플랫폼" 초기부터 참여, 서버 운영 및 보안 실무 경험
  • OpenStack, OpenShift, Docker, K8S 환경의 의료연구시스템 취약점 분석
  • 12개 병원 전용선 구축 및 네트워크 관리, VLAN 설정 및 이중화
  • 불필요한 수기 서버 점검을 자동화 스크립트로 개선하여 업무 시간 대폭 단축
03 · 스타트업

청주 이사 후 Flutter 개발자로

결혼을 앞두고 서울/경기의 보안 경력을 뒤로하고 청주로 이사, 서버와 네트워크 지식을 보유한 초급 개발자로서 경력을 시작했습니다.

다옴아이티에서 Delphi로 급여 정산 프로그램 유지보수와 Flutter CRM 개발을 병행하며, Golang 백엔드 RESTful API와 iOS/AOS 앱 출시까지 경험했습니다.

모다무에서는 블루투스 자동 주차 추적 어플리케이션을 개발, Flutter 블루투스 패키지 미지원 기능은 직접 Java/Swift로 네이티브 구현하여 출시와 운영까지 주도했습니다.

04 · 바이오톡스텍

GLP 전산 관리 & 자체 개발

GLP 기반의 비임상시험 CRO 전문업체에서 정보전산팀으로서 전산 시스템 관리부터 사내 솔루션 풀스택 개발까지 맡고 있습니다.

  • IACUC 동물윤리승인 웹 자체개발 (배포 후 무장애 유지)
  • EAS 대출관리시스템 Validation 완료 및 GLP 도입

05 · 마음의 말

달려온 발자취에 담아

간호사, 보안, 개발까지 새로운 환경과 문화 속에서 1인 이상의 역할을 하기까지 다양한 스트레스와 위기가 있었지만 더 나아질 것이라는 믿음과 인내와 함께 겸손하게 배우며 여기까지 달려오게 되었습니다.

이제는 비 전공자라는 타이틀을 벗어나 새로운 전공을 갖춘 IT인으로서 역량을 착실히 쌓아가며 업무에 성실히 임하고 있습니다.

좋은 사람들이 모인 곳이 좋은 기업이라고 생각합니다. 저도 좋은 기업을 함께 만들어가고 싶습니다.
